 = atribuição
 + adição
 - subtração
 * multiplicação
 / divisão
 % módulo

Unários:
+ valor positivo
- valor negativo
++ incremento de valor (de 1 em 1) [Colocar na frente na hora de imprimir para realizar o incremento imediato]
-- decremento de valor (de 1 em 1) [Colocar na frente na hora de imprimir para realizar o decremento imediato]
! negação [Colocar na frente na hora de imprimir para realizar o negação imediato]

Ternário:
? se a expressão for verdadeira
: se a expressão for falsa
ex: String resultado = a==b ?"verdadeiro" : "falso";

Relacionais:
== se uma variável é igual a outra
!= se uma variável é diferente da outra
> se uma variável é maior que a outra
< se uma variável é menor que a outra

.equals para quando temos conteúdos de dois objetos
ex: System.out.println(nomeUm.equals(nomeDois));

Lógicos:
&& é o operador lógico "E"
|| é o operador lógico "OU"
